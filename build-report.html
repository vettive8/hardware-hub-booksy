05 · Business logic

Small state machine, explicit guards.

AvailableRent when safe
In UseBound to employee
RepairUnavailable
Available → In Use → Available. Damaged return → Repair. Damage clears only with a service note.

Blocked

  • Rent damaged, Repair, or In Use
  • Return another user's rental
  • Delete/repair In Use hardware
  • Clear damage without evidence
Concurrency

Compare-and-set

UPDATE hardware SET status='In Use'
WHERE id=? AND status='Available'
AND is_damaged=0;
Authorization
